5. **05-database.mdc** - Database schema and operations
6. **06-weather-api.mdc** - Weather API integration guidelines
7. **07-frontend-features.mdc** - Pages, components, and state management
8. **08-error-handling.mdc** - Error handling patterns
9. **09-performance.mdc** - Performance considerations
10. **10-security.mdc** - Security best practices
11. **11-testing.mdc** - Testing guidelines
12. **12-hosting.mdc** - Hosting and deployment
13. **13-development-workflow.mdc** - Development workflow
14. **14-ui-ux.mdc** - UI/UX guidelines
15. **15-code-quality.mdc** - Code quality standards

## Usage

These files can be referenced individually or combined as needed.
